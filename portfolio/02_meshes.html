Project Procedural Geometric Modeling

3D Graphics

Project Overview

This project implements procedural geometry generation for various 3D shapes, including planes, cubes, spheres, toruses, cylinders, and cones. The aim was to construct these shapes algorithmically using vertices, indices, and math functions, avoiding hand-modeled geometry.

Tasks Completed

  • Implemented a generalized plane and index buffer generator for procedural surfaces.
  • Created 3D parametric shapes: Sphere, Torus, Cylinder, and Cone using trigonometric functions.
  • Implemented a custom lines-pattern generator to render meshes in wireframe mode dynamically.
  • Integrated the procedural meshes into an interactive WebGL demo using OpenGL abstractions.

Reflection

Generating complex 3D shapes parametrically was an excellent exercise in applied mathematics for graphics. Ensuring that index buffers correctly connected grid points required careful attention to topology. Implementing caps for the cylinder and cone shapes using the `add_cap` function was particularly challenging but ultimately rewarding when the shapes rendered flawlessly.
